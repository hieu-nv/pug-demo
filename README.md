# Pug Demo
## This is a project to create some examples of pug.

## Development
```bash
cd /path/to/project
yarn add global standard
yarn install
```

### How to run nodejs app with ES6
- Install babel
```bash
yarn add --dev babel-cli babel-preset-env
```
- Run nodejs app with ES6
```bash
babel-node --presets env index.js
```
